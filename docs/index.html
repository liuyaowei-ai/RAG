轻量级医疗 RAG 问答系统

本项目用于计算机复试展示，目标是实现一个可信、可解释、可本地部署的医疗知识问答系统。

Python 3.9 LangChain ChromaDB text2vec-base-chinese OpenAI / Ollama Streamlit

系统流程

  • 读取 PDF/TXT 医疗文档，进行分块切分。
  • 使用中文向量模型生成 Embedding，并存入本地 ChromaDB。
  • 用户提问后执行 Top-K 检索，再将上下文交给 LLM 生成回答。
  • 回答区展示“参考的本地文档来源”，提高可信度。

关键约束

模型被提示为“只能基于检索上下文回答”。若上下文无法支持回答，系统会返回：知识库中未找到相关信息，减少医疗场景下的幻觉。

运行方式

  • 构建向量库：python -m core.data_loader
  • 启动前端：streamlit run app.py
